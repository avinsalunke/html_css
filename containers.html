Types of Container<1>
  1. Block Level Container

    2. -- This type of container takes up the full width of their parent element and create new line after that which create vertical stack.
    examples. : div, section, header, footer, nav, article, aside.
    These elemetns are often use to create layout structure.
     
  2. Inline Contianer

    3. -- Inline emements do not create a new line after the element and they will only take the space needed by the content within them.
    examples : span, em, b, strong, a, label, button etc.
    These elements are often used for grouping & stylling the inline content such as text or inline-level elements.
     
  3. Generic Container
