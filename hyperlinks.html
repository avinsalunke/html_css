Target Attributes

1. _self : link will be open in the same frame or window of the web page. This is default behavior
2. _blank : link will be loaded to new window of the browser.
3. _parent : link will be loaded in the parent frame/window of the current web page, if the current webpage is in frameset or iframe.
4. _top : link will be loaded in the top-level browsing context, replacing any frames or iframes that may be present.
